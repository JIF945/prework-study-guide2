HTML

  • The body elements represents teh visible conttent shown to the user.
  • HTML stands for for hypertext markup language.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside and element.
  • color can be # + numbers, the word, or hex numbers.
  • CSS stands for cascading style sheets.

Git

  • Git status: checks what branch we are currently on.
  • Git checkout -b branch-name: creates a new branch and switches to it.
  • Git add -A: adds mods to a working branch.

JavaScript

  • A variable is a named container that allows us to store date in our code
  • Control flow is the order is which a computer executes code in a script
